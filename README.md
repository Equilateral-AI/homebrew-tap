# Equilateral-AI Homebrew Tap

Homebrew formulas for [Seawater](https://seawater.io) tools.

## Install

```bash
brew install Equilateral-AI/tap/morey
```

## Available formulas

- **morey** — Compiled governance daemon for AI agents. See [seawater.io/morey](https://seawater.io/morey/).
